# FastAPI Backend — Code Examples

Sanitized examples from production dashboard APIs and warehouse management systems built with FastAPI + SQL Server.

## Files

| File | What it demonstrates |
|------|---------------------|
| `inventory_router.py` | RESTful API: dynamic WHERE builder, whitelist-based sorting (anti-SQL-injection), pagination, Pydantic responses |
| `inventory_queries.py` | SQL query patterns: window functions, CROSS JOIN aggregation, OFFSET/FETCH pagination, CTE-based KPIs |
